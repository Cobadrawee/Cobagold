# COBA Gold - Redesigned Website

A premium, beautifully animated redesign of the COBA Token (gold-backed cryptocurrency) website, inspired by the elegant animations of [petenottage.co.uk](https://petenottage.co.uk/).

## Features

- **Gold-themed animated orb** – A stunning central visual with rotating orbits and scanning effects
- **Scroll-triggered animations** – Sections fade and slide into view as you scroll (Framer Motion)
- **Constellation background** – Subtle animated gold particles in the FAQ section
- **Premium dark theme** – Sophisticated gold/amber color palette for a serious financial aesthetic
- **Responsive design** – Works seamlessly on desktop and mobile
- **All original COBA content** – Token metrics, roadmap, FAQ, how it works, and contact info preserved

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** – Fast build tool
- **Framer Motion** – Smooth animations
- **Tailwind CSS v4** – Utility-first styling
- **DM Sans** & **Playfair Display** – Typography

## Getting Started

```bash
# Install dependencies (if not already done)
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

Open [http://localhost:5173](http://localhost:5173) to view the site.
